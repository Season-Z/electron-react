# electron-react

## Introduce

- electron@16
- electron-builder@22
- react@17
- react-router-dom@6
- typescript@4
- webpack@5

tpescript语法编写，webpack实现打包主进程和渲染进程，electron热重启加载。

## start

```js
yarn

yarn start

yarn build
```
